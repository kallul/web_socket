# WebSocket Chat Application
A real-time bidirectional chat application built with Python using WebSockets. This application demonstrates how to create a simple 
chat system where multiple clients can connect to a server and communicate with each other in real-time.

## Features
a) Real-time bidirectional communication between server and clients
b) Multiple client support with message broadcasting
c) Server can broadcast messages to all connected clients
d) Proper error handling for connection issues
e) Clean terminal UI with emoji indicators
f) Graceful disconnection handling
